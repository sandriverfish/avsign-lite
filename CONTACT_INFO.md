## Email Best Practices

### When Emailing Support

**Include:**
1. **Platform**: Windows, Android, or Linux
2. **Version**: v1.0.0-beta (or your version)
3. **Issue Description**: What happened?
4. **Steps to Reproduce**: How can we recreate it?
5. **Screenshots**: If applicable
6. **Error Messages**: Copy/paste any errors

**Example Good Email:**
```
Subject: [BUG] Upload fails on Windows 11

Platform: Windows 11 Pro
Version: v1.0.0-beta

Issue: When I try to upload a video file, the upload fails at 50%.

Steps:
1. Go to Resources > Upload
2. Select MP4 file (250 MB)
3. Click Upload
4. Progress bar reaches 50% then stops

Error message: "Upload failed: Connection reset"

Screenshot attached.

Thanks!
```
