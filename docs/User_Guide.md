## Quick Reference

### Key Concepts

**Resources**: Media files (images, videos) uploaded to avSign.lite
**Layouts**: Screen designs with defined zones
**Zones**: Areas within a layout where content appears
**Programs**: Playlists of content assigned to layouts
**Terminals**: Physical display devices

### Supported File Types

**Images**: JPG, PNG, GIF, BMP, WebP
**Videos**: MP4 (recommended), AVI, MKV, MOV, WebM

## Workflow Guide

### 1. Resource Management

Upload and organize your media files.

**Best Practices:**
- Keep images under 10 MB
- Keep videos under 500 MB
- Use descriptive file names
- Delete unused files regularly

**See:** [Quick Start - Upload an Image](Quick_Start.md#step-1-upload-an-image-1-minute)

---

### 2. Layout Design

Create screen layouts with zones.

**Options:**
- **Use Templates**: Quick start with pre-built layouts
- **Custom Layouts**: Design from scratch

**Available Templates:**
- Full Screen (single zone)
- Picture-in-Picture (main + overlay)
- Side-by-Side (two zones)
- Quad Dashboard (four zones)
- Video Wall (multi-screen)

**See:** [Quick Start - Create a Layout](Quick_Start.md#step-2-create-a-layout-2-minutes)

---

### 3. Program Creation

Build playlists of content.

**Steps:**
1. Create program
2. Select layout
3. Add content to each zone
4. Set durations
5. Save

**Features:**
- Multiple items per zone (slideshow)
- Custom durations per item
- Automatic looping

**See:** [Quick Start - Create a Program](Quick_Start.md#step-3-create-a-program-2-minutes)

---

### 4. Device Management

Register and manage display devices.

**Terminal Setup:**
1. Add terminal
2. Set name and resolution
3. Save

**Publishing:**
1. Select program
2. Click "Publish"
3. Choose terminal(s)
4. Confirm

**See:** [Quick Start - Add a Terminal](Quick_Start.md#step-4-add-a-terminal-1-minute)

## Tips & Best Practices

### Content Optimization
- Compress images before uploading
- Use MP4 format for videos
- Match content to display resolution
- Keep file sizes reasonable

### Layout Design
- Avoid overlapping zones
- Use consistent aspect ratios
- Fill entire screen
- Test on actual display

### Performance
- Limit playlists to 20 items
- Clear cache regularly
- Restart displays weekly
- Monitor storage space

### Organization
- Use descriptive names
- Tag related content
- Delete old files
- Regular backups
